# Meta Description - Homepage

## Prompt (Hungarian)

Írj egy SEO-optimalizált meta leírást a {{business.name}} kezdőlapjához.

A cég a {{business.industry}} iparágban tevékenykedik, szolgáltatások: {{services_list}}.
Hely: {{business.address.city}}.

A meta leírás legyen:
- Pontosan 150-160 karakter
- Tartalmazza a fő kulcsszavakat: {{business.industry}}, {{business.address.city}}
- Cselekvésre ösztönző (call-to-action)
- Magyar nyelven

## Output Format

- Single line meta description
- 150-160 characters
- Include location and industry keywords
- Hungarian language

## Variables Required

- `business.name` - Company name
- `business.industry` - Industry
- `services_list` - All services
- `business.address.city` - City
