# Image Assets

## Quick Start

1. **Add original images** to `originals/` folder
2. **Run optimization script**: `python scripts/optimize-images.py`
3. **Build the site** - optimized images are automatically used

## Folder Structure

```

├── originals/              ← PUT YOUR IMAGES HERE
│   ├── hero-bg.jpg
│   ├── logo.png
```

## Required Images

### Logo & Branding

| File | Recommended Size | Notes |
|------|-----------------|-------|
| `originals/logo.png` | 400x130px+ | PNG with transparency |
| `originals/og-image.jpg` | 1200x630px | Social media sharing |

### Hero Background

| File | Recommended Size | Notes |
|------|-----------------|-------|
| `originals/hero-bg.jpg` | 2000x860px+ | High quality, will be cropped to 7:3 |

### Service Images

| File | Recommended Size | Notes |
|------|-----------------|-------|
| `originals/services/[slug].jpg` | 800x533px+ | Will be cropped to 3:2 |

**Example filenames:**
- `services/konyhabutor-keszites.jpg`
- `services/beepitett-szekreny.jpg`
- `services/gardrob-keszites.jpg`

### Team Photos (Optional)

| File | Recommended Size | Notes |
|------|-----------------|-------|
| `originals/team/[slug].jpg` | 400x400px+ | Square, will be cropped 1:1 |

**Example filenames:**
- `team/kovacs-janos.jpg`
- `team/nagy-peter.jpg`

## Optimization Script

### Requirements

```bash
pip install Pillow
```

### Usage

```bash
python scripts/optimize-images.py
```

### What It Does

1. **Reads** images from `originals/`
2. **Crops** to correct aspect ratios
3. **Resizes** to multiple sizes (srcset)
4. **Converts** to WebP + JPG fallback
5. **Outputs** to main `assets/images/` folder

### Generated Sizes

| Type | Sizes (width) | Aspect Ratio |
|------|---------------|--------------|
| Hero | 400, 800, 1200, 1600, 2000px | 7:3 |
| Services | 300, 400, 600, 800px | 3:2 |
| Team | 150, 200, 300, 400px | 1:1 |
| Logo | 100, 150, 200, 300px | Original |
| OG Image | 1200px | 1200:630 |

## Image Guidelines

### Quality

- Use high-quality source images (larger is better)
- The script will optimize automatically
- Minimum recommended: 800px width for services

### Content

- Service images should clearly represent the work
- Avoid generic stock photos
- Real project photos work best for local businesses
- Ensure good lighting and composition

### Aspect Ratios

The script automatically crops images to the correct ratio:

| Image Type | Aspect Ratio | Example |
|------------|--------------|---------|
| Hero | 7:3 | 1400x600 |
| Services | 3:2 | 600x400 |
| Team | 1:1 | 400x400 |
| OG Image | 1200:630 | 1200x630 |

## Browser Support

The templates use `<picture>` elements with:

- **WebP**: Modern browsers (Chrome, Firefox, Edge, Safari 14+)
- **JPEG fallback**: Older browsers
- **srcset**: Responsive image loading
- **sizes**: Optimal size hints

## Troubleshooting

### Script not finding images?
- Check images are in `originals/` folder
- Supported formats: JPG, PNG, WebP, GIF

### WebP not showing?
- Check browser supports WebP
- Fallback JPG should load automatically

### Images look stretched?
- Source image aspect ratio may differ
- Script crops from center - check composition

### Script errors?
```bash
pip install --upgrade Pillow
```
